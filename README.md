# beatmaker
An application resembeling a drum machine with classic 909 house sounds

## Technologies Used
- JavaScript
- HTML 5
- CSS3

## Live Demo


## Features
- User can click on pads to start a sequence
- User can choose different drum sounds
- User can adjust tempo
- User can press play/pause
